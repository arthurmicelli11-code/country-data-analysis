# Country Data Analysis

## Project Description
Personal project analyzing country level datasetsto explore trends between them like GDP, population, education, and others.

## Steps
1. Collected and cleaned public datasets with country-level indicators.
2. Planned analysis using Python and SQL to explore correlations between variables.
3. Designed dashboards and charts in Google Sheets to visualize insights.
4. Applied statistical methods learned from Stanford Statistics and Google Data Analytics courses.
5. Practiced combining multiple data sources to generate meaningful insights.

## Technologies
- Python (Pandas)
- SQL
- Google Sheets
- Tableau (basic)
- Statistics
